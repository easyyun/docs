私有化部署

  • 私有独立部署版（付费），提供源码和后期更新功能，私有化版本API接口和云端API接口通用，只需改少量配置即可启动，易装易用。
  • 对于数据隐私有需求的企业或个人，公众号留言，我们提供1对1部署等服务。
  • 独立部署后，基于 API 可自行开发更多功能，比如小程序，APP，网页等等，没有任何功能的限制。

服务器要求

最低要求：2核 CPU ，2G 内存。

部署步骤

  • 找官方获取源码
  • 项目代码根目录 cp .env.example .env
  • 配置 .env 里面的OSS 配置，支持腾讯云COS、阿里云OSS
  • cp app_key.example.py app_key.py
  • ./start.sh

功能接口测试

curl -X 'POST' \
  'http://localhost:8000/v1/pdf/pdf_to_image' \
  -H 'accept: application/json' \
  -H 'Content-Type: application/json' \
  -d '{
  "app_key": "app_key_test",
  "input": "https://www.pdfai.cn/static/example/files/four_pages.pdf",
  "page": "1-N",
  "quality": "high"
}'
